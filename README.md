# Hostplus
最新可用hosts科学上网

##特性
*   google等都是支持的，你懂的
*   ipv6油管
*   https重定向
*   更新提醒

##使用
请下载[最新版](https://github.com/banben/Hostplus/releases)，度盘下载[点这](http://pan.baidu.com/s/1ge5u9Cf)

##FAQ
###Hostplus支持Mac吗?
暂时只支持windows

###Hostplus突然不能用了?
在系统托盘找到Hostplus右键选修复Hosts，重启计算机。如果使用了SwichySharp插件，请使用系统代理
