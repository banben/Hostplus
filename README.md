# Hostplus
全自动hosts科学上网

##特性
*   google等都是支持的，你懂的
*   ipv6油管
*   https重定向
*   更新提醒

##使用
从[Github](https://github.com/banben/Hostplus/releases)下载，从[镜像](http://123.206.16.184/Hostplus-1.0.2.zip)下载，运行后右键点托盘图标选择修复hosts

##FAQ
###Hostplus支持Mac吗?
暂时只支持windows

###Hostplus突然不能用了?
在系统托盘找到Hostplus右键选修复Hosts，重启计算机。如果使用了SwichySharp插件，请使用系统代理
