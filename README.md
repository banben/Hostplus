# Hostplus
hosts线路优化器，通过hosts文件和http代理重定向优化线路，总之没时间解释了，快上车。实验性版本，欢迎反馈

*   Youtube ipv6 支持
*   Google 全家桶支持
*   Facebook支持
*   Dropbox支持
*   Wikipedia支持
*   其它支持

##FAQ
###Hostplus支持Mac吗?

###Hostplus突然不能用了?
在系统托盘找到Hostplus右键选修复Hosts，重启计算机。如果使用了SwichySharp插件，请使用系统代理
